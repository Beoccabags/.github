## Over ons

Beocca is een klein familiebedrijf van Sven en Lisa. We maken tijdloze leren tassen, portemonnees en accessoires die meegaan met je leven en mooier worden met de jaren.

Ons leer is plantaardig gelooid en elk product wordt met de hand afgewerkt in ons eigen atelier.

## Wat hier op GitHub leeft

Deze organisatie bevat de tooling achter onze webshop en marketplaces:

- Shopify theme customizations (Clutch)
- Marketplace integraties (Bol.com, Amazon SP-API, ChannelEngine)
- Moneybird automatiseringen voor bookkeeping
- Klaviyo email templates en flows
- Productfeed en image processing scripts
